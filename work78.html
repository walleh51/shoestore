1. What is HTML and what does it stand for?

HTML stands for HyperText Markup Language. It is the standard language used to create and design documents on the web. HTML defines the structure and layout of a web page using a system of tags and attributes.

2. What is the difference between HTML and XHTML?

HTML (HyperText Markup Language) is a markup language used to create web pages, while XHTML (Extensible HyperText Markup Language) is a more strict version of HTML that adheres to XML syntax rules. XHTML requires all tags to be properly closed and nested, making it more rigorous than HTML.
